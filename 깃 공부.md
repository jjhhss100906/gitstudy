1. 프로젝트 시작 및 가져오기

 새로운 프로젝트를 시작하거나 기존 프로젝트를 내 컴퓨터로 가져올 때 쓰는 명령어이다.
 * git init   
 설명 : 현재 폴더를 git이 관리하는 저장소로 초기화한다.     
 사용법 : ```git init```
 * git clone    
 설명 : GitHub에 있는 원격 저장소를 그대로 내 컴퓨터로 복사해 온다.   
 사용법 : ```git clone 원격 저장소 url```

2. 작업하고 저장

* git status   
설명 : 현재 어떤 파일이 수정되었고, 어떤 파일이 장바구니에 담겼는지 상태를 보여줍니다.    
사용법 : ```git status``` 
* git add     
설명 : 수정된 파일을 커밋하기 전에 장바구니에 올리는 명령어입니다.    
사용법 :```git add <파일명>```(특정 파일만 담기) , ```git add .```(변경된 모든 파일 한 번에 담기)
* git commit     
설명 : 장바구니에 담은 파일들을 하나의 스냅샷으로 기록하고 저장합니다.    
사용법 : ```git commit -m "커밋 메시지" ```
* git log    
설명 : 지금까지 만든 커밋들의 히스토리를 확인합니다.     
사용법 : ```git log```

3. 브랜치 관련 명령어     

* git branch
설명 : 현재 존재하는 브랜치 목록을 보여주고,새로운 브랜치를 생성한다.    
사용법 : ```git branch```(목록 확인) , ```git branch <새브랜치이름>```(새로운 브랜치 만들기)    
* git checkout or git switch
설명 : 다른 브랜치로 이동한다.     
사용법 : ```git checkout <브랜치이름>```(이동하기) , ```git checkout -b <새브랜치이름>```(브랜치를 만들면서 동시에 이동하기)     

4. github와 동기화하기
* git push
설명 : 내 컴퓨터에서 커밋한 내용을 github로 업로드한다.    
사용법 : ```git push origin <브랜치이름>```
* git pull
설명 : github에 있는 최신 변경사항을 다운로드하여 내 컴퓨터 코드에 바로 합친다.      
사용법 : ```git pull origin <브랜치이름>```
